

# Green Grass Store

Premium home and garden e-commerce storefront for the UAE market.

## About

Green Grass Store is a modern e-commerce platform specializing in plants, pots, vases, and home décor products. The platform is built with React, TypeScript, Supabase, Zustand, and Tailwind CSS, and is designed for a seamless, dynamic, and admin-controllable shopping experience.

## Key Features & Functionality

- **Dynamic Product Catalog:**
	- Products are managed in the Supabase backend and displayed dynamically on the frontend.
	- Real-time updates for product stock, price, and details.

- **Shopping Cart & Checkout:**
	- Full shopping cart with add/remove/update quantity.
	- Checkout supports multiple payment methods: Cash on Delivery, WhatsApp Order, Online Payment, and Direct Bank Transfer (wire transfer/direct deposit).
	- Coupon/discount code support with validation and usage tracking.
	- Order summary, shipping calculation, and order confirmation with unique order number.

- **Admin Dashboard:**
	- Manage featured categories: activate/deactivate, reorder (drag & drop), set images, and control product count per category.
	- Manage homepage gift section: manually select and order gift products, or fallback to dynamic filtering.
	- Manage payment methods: enable/disable, set bank details, instructions, and payment banners.
	- Real-time updates for all admin changes reflected instantly on the homepage and checkout.

- **Multi-language Support:**
	- English and Arabic language support throughout the site.
	- All product, category, and UI text is localized.

- **Wishlist & Compare:**
	- Users can add products to wishlist and compare lists.

- **Customer Accounts:**
	- User registration, login, and profile management.
	- Order history and tracking.

- **WhatsApp Order:**
	- Customers can place orders directly via WhatsApp with pre-filled order details.

- **Responsive Design:**
	- Fully responsive and mobile-friendly UI.

- **Other Features:**
	- SEO-friendly routing and meta tags.
	- Framer Motion animations for smooth UI transitions.
	- Drag-and-drop reordering for admin sections.
	- Secure payment and easy return banners.
	- Newsletter subscription and popups.

## Technology Stack

- React 18 with TypeScript
- Tailwind CSS for styling
- Supabase (Postgres) for backend/database
- Zustand for state management
- Framer Motion for animations
- @hello-pangea/dnd for drag-and-drop

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
